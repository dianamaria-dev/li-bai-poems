# The Wine Poet — Li Bai Poetry Website


A bilingual (English/Romanian/Chinese) website dedicated to the poems of Li Bai (701–762 AD), the legendary "Immortal Poet" of the Tang Dynasty.

## Features

- **7 famous poems** by Li Bai with complete Chinese text and translations
- **Daily poem** feature with "Next Poem" navigation
- **Language switcher** — toggle between English, Romanian, and Chinese
- **Favorites system** — save poems to localStorage
- **Calligraphy images** — original hand-drawn calligraphy for each poem
- **Sliding wine-red menu** — responsive design for mobile and desktop
- **Sakura petals animation** — subtle falling cherry blossoms

## Pages

| Page | Description |
|------|-------------|
| Home | Daily poem with calligraphy image and star button |
| All Poems | List of all 7 poems with previews |
| Poem Detail | Deep dive with full poem, commentary, and navigation |
| My Favorites | Saved poems from localStorage |
| About Li Bai | Biography + contact information |

## Technologies Used

- HTML5
- CSS3 (Flexbox, Grid, animations)
- Vanilla JavaScript
- LocalStorage API
